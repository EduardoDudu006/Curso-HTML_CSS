flex-direction

As configurações já são flex-direction: row;
Alinhamento horizontal da esquerda para direita
Também podemos inverter para flex-direction: row-reverse;
Alinhamento horizontal da direita para esquerda
Mas, é possível alterar essa configuração para flex-direction: column;
Alinhamento vertical de cima para baixo
Também podemos inverter para flex-direction: column-reverse;
Alinhamento vertical de baixo para cima

Eixos (axis)

No flex-direction: row; é criado um main-axis(eixo principal)
Onde são criados dois pontos, main-start (esquerda) e main-end (direita)
No flex-direction: column; é criado o cross-axis (eixo transversal)
Onde são criados dois pontos, cross-start (cima) e cross-end (baixo)

Quando utilizamos o flex-direction: row;
São criados dois pontos, main-axis (esquerda / direita) e cross-axis (cima / baixo)
Já no flex-direction: row-reverse;
São criados dois pontos, main-axis (direita / esquerda) e cross-axis (cima / baixo)
Os dois pontos, cross-start (cima) e cross-end (baixo) permanecem na mesma posição

Quando utilizamos o flex-direction: column;
São criados dois pontos, main-axis (cima / baixo) e cross-axis (esquerda / direita)
Já no flex-direction: column-reverse;
São criados dois pontos, main-axis (baixo / cima) e cross-axis (esquerda / direita)
