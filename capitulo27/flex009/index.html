Aula 09: Controle de tamanho de itens flexiveis


Vamos conhecer as propriedades flex-shrink: (Pode Encolher?) e flex-grow: (Pode Aumentar?)
Por padrão essas duas propriedades vêm com:
flex-shrink: 1 - Isso significa que o elemento pode encolher.
flex-grow: 0 - Isso significa que o elemento não pode aumentar.

Exemplo 01:

.sessao01 {
width: 550px;
display: flex;
flex-flow: row nowrap;
}
.item01 {
background-color: aquamarine;
border: 1px solid darkgreen;
text-align: center;
padding: 7px;
flex-basis: 150px;
}

Exemplo 09 - flex-basis:, flex-grow:, flex-shrink:

width: 950px;
display: flex;
flex-flow: row nowrap;
}
.item09 {
flex-basis: 150px;
}
#i1 {
flex-basis: 150px;
flex-grow: 0; (O item Não pode Crescer)
flex-shrink: 1; (O item pode Encolher)
}
#i2 {
flex-basis: 150px;
flex-grow: 2; (O item B pode Crescer o dobro em relação ao item C)
flex-shrink: 0; (O item Não pode Encolher)
}
#i3 {
flex-basis: 150px;
flex-grow: 1;
flex-shrink: 2; (O item C pode Encolher o dobro em relação ao item A)
