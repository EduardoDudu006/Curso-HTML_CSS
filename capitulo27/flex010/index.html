Aula 10: A Propriedade Flex


Vamos conhecer a propriedade flex:

É uma propriedade que se aplica aos flex-items, não ao flex-container
Essa propriedade funciona como o Shorthand configurando numa única linha...
O flex--grow, flex-shrink e o flex-basis.

Exemplo:
flex-basis: 150px;
flex-grow: 0;
flex-shrink: 1;

Pode ser substituido por apenas:
flex: 0 1 150px;


Fórmula:

flex = flex-grow + flex-shrink + flex-basis

Obs.: Sempre Nesta Ordem

Exemplo 01 - flex-basis:, flex-grow:, flex-shrink:

.sessao01 {
width: 950px;
display: flex;
flex-flow: row nowrap;
}
#i1 {
flex-basis: 150px;
flex-grow: 0; (O item Não pode Crescer)
flex-shrink: 1; (O item pode Encolher)
}
#i2 {
flex-basis: 150px;
flex-grow: 2; (O item B pode Crescer o dobro em relação ao item C)
flex-shrink: 0; (O item Não pode Encolher)
}
#i3 {
flex-basis: 150px;
flex-grow: 1;
flex-shrink: 2; (O item C pode Encolher o dobro em relação ao item A)
}

Obs.: Sempre Nesta Ordem.

A configuração Padrão é:

flex: 0 1 auto; ou
flex: initial;

Configuração para deixar os elementos Não Flexiveis:

flex: 0 0 auto; ou
flex: none;

Configuração para que os elementos tenham o Máximo de Flexibilidade:

flex: 1 1 auto; ou
flex: auto;

Configuração para que modificar o valor do flex-grow:

flex: 1; flex: 2; flex: 3; ou
flex: 1 1 auto; flex: 2 1 auto; flex: 3 1 auto;
