✅ Span

Caixa13 {

}

Caixa15 {

}

Propriedade Span:

Define a Expansão da Área
No Exemplo acima: (caixa13)
O Grid-row vai da linha 1
Até a linha 4.
Então a Expansão foi de 3 linhas.
Já o Grid-Column vai da Coluna 1
Até a Coluna 3.
Então a Expansão foi de 2 Colunas.

grid-area: 1 / 1 / span 3 / span 2;
