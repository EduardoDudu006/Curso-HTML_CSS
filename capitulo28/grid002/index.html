A Anatomia de um Grid

No CSS Grid, a relação entre esses dois conceitos é de "pai e filho". Imagine que o Grid Container é a caixa externa e os Grid Items são o conteúdo dentro dela.

Grid Container
É o elemento pai onde você define a estrutura da grade. Para criá-lo, você aplica display: grid ou display: inline-grid a um elemento HTML. O que ele faz: Controla o número de colunas, o tamanho das linhas, o espaçamento (gap) e o alinhamento geral. Principais propriedades: grid-template-columns, grid-template-rows, gap, justify-items, align-items.
