Aula 01: O Que é Grid Layout?

Grid Layout e Flex Layout são bem parecidos, porém não são Iguais, devem ser utilizados em momentos e por motivos diferentes.

W3C
A especificação do Grid Layout surgiu em 2011.
(CSS Grid Layout Module Level 1)
Definição: "CSS Grid Layout é um sistema para criação de layout baseado em uma grade bidimencional e otimizado para design de interface de usuário. Nesse modelo de grade, os elementos-filhos do container que defina a grade podem ser posicionados livremente em espaços criados na estrutura, quer ela tenha sido definida com suas dimenções flexíveis ou fixas."
