Propriedades:
  • display
  • grid-template-row
  • grid-template-columns
  • grid-gap

Configurações CSS:

.container {
display: grid;
grid-template-columns: auto auto;
}
Essa Configuração estabelece que o Grid-Container terá 2 Colunas.
