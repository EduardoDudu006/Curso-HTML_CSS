Principais formatações

Negrito / Destaque

Nessa frase temos um - Termo em Negrito - usando a tag <b> (Não Semântica).

Nessa frase, tenho um - Termo em Destaque - usando a tag <strong>(Semântica).

Itálico / Ênfase

Nesta frase, temos um - Termo em Itálico - usando a tag <i> (Não Semântica).

Nesta frase, temos um - Termo em Ênfase - usando a tag <em> (Semântica).


Texto Marcado

Podemos criar também um - texto marcado - usando a tag <mark> (Não Semântica).

Podemos criar também um - - texto marcado usando a tag <mark> com o parâmetro CSS (Semântica).


Texto Grande e Pequeno

Estamos criando um - Texto Grande - usando a tag <big> (Obsoleta).

Estamos criando um - Texto Pequeno - usando a tag <small> (Não Obsoleta).

Podemos marcar um Texto como Excluído para indicar que ele deve ser lido , mas não considerado usando a tag <del> (Semântica).

Podemos marcar um Texto como Inserido / Sublinhado para dar ênfase e/ou indicar que ele foi adicionado depois usando a tag <ins> (Semântica).

Existe também o Texto Sublinhado com a tag <u> (Não Semântica).


Texto Sobrescrito

Para inserir coisas do tipo x20+3 - usando a tag <sup> (Semântica).


Texto Subscrito

Para inserir coisas do tipo H2O - usando a tag <sub> (Semântica).
